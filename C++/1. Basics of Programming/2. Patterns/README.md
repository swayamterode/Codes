### 1.Pattern 1

<details>
 <summary>Click to Expand!</summary>
 
#### Format
 
``` text
1. You are given a number n.
2. You've to create a pattern of * and separated by tab as shown in output format.
```
 
#### Input Format
 
 ```text
A number n
 ```
 
#### Constraints

 ```text
 1 <= n <= 44
 ```
 
#### Sample Input

 ```text
 5
``` 
 
#### Output Format

```text
*	
*	*	
*	*	*	
*	*	*	*	
*	*	*	*	*
```
